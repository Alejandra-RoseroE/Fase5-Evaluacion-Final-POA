## 📝 Descripción del Proyecto
Este repositorio contiene la solución algorítmica al **Problema 2** del banco de problemas de la Fase 5. 

El programa simula un sistema de gestión de precios para el menú de un restaurante. A través de la evaluación de una matriz de datos, el sistema procesa el catálogo de productos y aplica una regla de negocio específica: **otorgar un 15% de descuento** a los productos que pertenezcan a una categoría determinada y cuyo precio base supere un umbral establecido.

---

## 🛠️ Requisitos Técnicos Implementados
El código fuente fue desarrollado bajo el paradigma de programación estructurada, cumpliendo estrictamente con los criterios de la rúbrica de evaluación:

* ✅ **Lenguaje de Programación:** Python puro (sin uso de librerías externas).
* ✅ **Matrices (Arreglos):** Implementación de una matriz bidimensional (lista de listas) con 7 registros. Estructura: `[Nombre del Producto, Categoría, Precio Base]`.
* ✅ **Estructuras de Control:** Uso de condicionales lógicos (`if` / `else`) para evaluar si se cumplen las condiciones del descuento.
* ✅ **Estructuras de Repetición:** Iteración automatizada mediante un bucle `for` para recorrer todos los elementos de la matriz.
* ✅ **Módulos / Funciones:** Lógica de negocio encapsulada y modularizada en la función `calcular_precio_final()`.
